# Express | Social Web API

Create a `.env` file for enviroments variables

```sh
PORT=3000
DB_USER='username'
DB_PASSWORD='password'
DB_HOST='host of postgres'
DB_PORT='postgres port'
DB_NAME='database name'
JWT_SECRET=<secret-jwt>
```

Run project
```bash
npm start
```
Run in development mode
```bash
npm run dev
```
